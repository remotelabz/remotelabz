RemoteLabz
==========

This project is the 2nd version of [RemoteLabz](www.remotelabz.fr) built with [Symfony](https://symfony.com/) and [ReactJS](https://reactjs.org/).

Documentation is available at https://docs.remotelabz.com/.

We recommand to install the branch [Upgrade-2.5](https://github.com/remotelabz/remotelabz/tree/Upgrade-2.5).
